# Domains Leyu Lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为通用 HTML 页面资源的存储与展示空间。

## 项目简介

本仓库的主要目的是收集、整理并托管各类独立的 HTML 页面。这些页面可能来源于不同的项目、实验或临时需求，通过统一存放的方式便于版本管理与访问。

## 目录结构

```
domains-leyu-lab/
├── pages/          # 存放各个 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件对应一个独立的 HTML 页面。
- `assets/` 目录用于存放页面引用的资源文件，保持页面与资源的相对路径关系。

## 页面归档说明

- 所有 HTML 页面均以独立文件形式存放，不依赖外部数据库或后端服务。
- 页面内容可能包含实验性代码、临时展示内容或静态信息。
- 每个页面通常自包含或仅引用本仓库内资源，以确保可离线查看。

## 使用方式

您可以直接通过 GitHub Pages 或其他静态托管服务访问本仓库中的 HTML 文件。如需在本地查看，请克隆仓库后使用浏览器打开对应页面文件。

```bash
git clone https://github.com/your-username/domains-leyu-lab.git
cd domains-leyu-lab
```

然后打开 `pages/` 目录下的任意 HTML 文件即可。
